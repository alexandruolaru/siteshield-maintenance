== Description ==
SiteShield Maintenance Mode is a custom maintenance mode plugin designed to enable flexible maintenance mode management for your WordPress website. It allows you to set a custom maintenance page title, HTML content, and selectively grant access to specific user roles while the site is under maintenance.

== Features ==
* Custom Maintenance Page Title
* Custom HTML Content
* Select which user roles can access the site while in maintenance mode
* Administrators always have access

== Installation ==
1. Upload the `siteshield-maintenance` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > SiteShield Maintenance Mode** to configure the plugin.

== Minimum Requirements ==
* WordPress 5.0 or higher
* PHP version 7.4 or higher

== Frequently Asked Questions ==

= How do I enable maintenance mode? =
Go to **Settings > SiteShield Maintenance Mode** and check the "Enable Maintenance Mode" option.

= Can I customize the maintenance page content? =
Yes, you can customize the title and the content of the maintenance page using the WordPress HTML editor.
